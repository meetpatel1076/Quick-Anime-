# Quick Anime

Quick Anime is a frontend web project designed to help anime lovers easily discover and compare different platforms to watch anime.

The goal of this website is to provide quick access to multiple anime sources in one place, making it easier for users to decide which platform best suits their needs.

---

## Features
- Clean and simple user interface
- Easy navigation between anime sources
- Responsive design for different screen sizes
- Fast access to popular anime platforms

---

## Technologies Used
- HTML
- CSS
- SVG & PNG assets
